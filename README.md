# Personal Website

A lightweight static personal website:

- `index.html` main page
- `styles.css` styling
- `script.js` theme toggle + smooth scroll

## Run locally

1. Open `index.html` in browser (or use Live Server extension)
2. Customize with your name, projects, and links
